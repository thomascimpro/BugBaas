# BugBaas 2.10.3

## Arcade balance and feel

- Bug Tower now changes to a new original world every 100 floors and shows floor numbers on every platform.
- Platform shrink, gaps, moving platforms and rising floor pressure scale smoothly over a much longer climb.
- Tower score is based on height and best combo, never survival time.
- Charged high-speed jumps can spin the animated beetle.
- Bubble Swarm only shoots colors still present on the board.
- Bubble aiming now has a visible dotted guide, one-wall bounce and a smooth projectile flight.
- Smaller bubbles and slower early pressure make the opening fairer; the solo run still becomes unwinnable over time.
- Nine original Tower and Bubble sound effects are included on Android, with matching synthesized web feedback.
