# Image generation prompts

Algemene stijl voor alle prompts:
`HD mobile game asset, friendly 3D cartoon fantasy bug world, polished game art, vibrant colors, soft lighting, clean shape language, no text, no watermark, no logo.`

Voor transparante objecten altijd toevoegen:
`transparent background, isolated subject, tightly cropped, full object visible, PNG-ready.`

Voor backgrounds altijd toevoegen:
`full-screen mobile game background, 16:9 safe composition, no text, no UI, no characters blocking center gameplay area.`
